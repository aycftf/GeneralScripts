This repo contains a world of finished / unfinished helper scripts with the goal of hardening the Linux Operating System.
Repo Contents:
- Autid Framework log automation, installation, parser, and dump. Tracks / Logs various linux syscalls, epbf, and general execution / capability properties surrounding system activities.
- Sysctl tunable automation; i.e. formatting / hardening / provisioning
- Find process PID, File Descriptor (mem map), and binary run location from name
- Azure Blob storage automation (Upload / Delete)
- Automatic external media encryption using LUKS and bash
- CLONEZILLA install -> bootable iso utility for backing up data
